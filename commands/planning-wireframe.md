# Planning Wireframe 워크플로우 실행

이 플러그인의 planning-wireframe 워크플로우를 사용한다.

수행 순서:

1. `README.md`, `skills/planning-wireframe/SKILL.md`, `skills/planning-wireframe/USAGE.md`, `skills/planning-wireframe/state-schema.md`를 확인한다.
2. 현재 세션이 있으면 `status`, 없으면 `init`을 사용한다.
3. 질문 수집은 `next`와 `answer`로 진행한다.
4. 문서 생성이 필요하면 `render-doc`를 실행한다.
5. Figma 연동이 필요하면 `figma-manifest`, `attach-screenshot`, `annotate`를 순서대로 사용한다.
6. 결과와 다음 명령을 한국어로 짧고 명확하게 정리한다.
